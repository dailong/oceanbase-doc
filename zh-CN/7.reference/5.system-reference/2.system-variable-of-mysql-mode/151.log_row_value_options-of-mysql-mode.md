# log_row_value_options

`log_row_value_options` 用于控制 Lob 是否使用部分更新。
|**属性**| **描述**|
|--------|---------|
|参数类型 |varchar  |
|默认值   | 空字符串 |
|取值范围 |['', 'partial_lob']<blockquote>**说明** <br> '' 代表空字符串。</blockquote>        |
|生效范围 |<ul><li>Global</li><li>Session</li></ul>|
|是否参与序列化|否                                  |
