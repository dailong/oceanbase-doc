stack_size 
===============================

`stack_size` 用于设置程序函数调用栈的大小。


|      **属性**      |    **描述**     |
|------------------|---------------|
| 参数类型             | 容量单位          |
| 默认值              | 512K            |
| 取值范围             | \[512K, 20M\] |
| 是否重启 observer 进程生效 | 是             |
