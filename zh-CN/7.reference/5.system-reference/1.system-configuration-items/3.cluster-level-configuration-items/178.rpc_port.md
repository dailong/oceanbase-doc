rpc_port 
=============================

rpc_port 用于设置远程访问的协议端口号。


|      **属性**      |     **描述**      |
|------------------|-----------------|
| 参数类型             | 整型              |
| 默认值              | 2882            |
| 取值范围             | \[1025, 65535\] |
| 是否重启 observer 进程生效 | 否               |
